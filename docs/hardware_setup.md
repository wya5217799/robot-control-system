# 硬件连接说明

## GPIO引脚连接

### 电机控制引脚
- **左电机**:
  - 引脚0 (WiringPi) -> 电机驱动器IN1
  - 引脚1 (WiringPi) -> 电机驱动器IN2
  
- **右电机**:
  - 引脚2 (WiringPi) -> 电机驱动器IN3
  - 引脚3 (WiringPi) -> 电机驱动器IN4

### 串口连接
- **USB转串口模块**:
  - 设备路径: `/dev/ttyUSB0`
  - 波特率: 9600
  - 数据位: 8
  - 停止位: 1
  - 校验位: 无

### 摄像头连接
- **USB摄像头**:
  - 设备编号: 0 (默认)
  - 支持标准UVC协议

## 电源要求

- **树莓派**: 5V 2.5A
- **电机**: 根据电机规格选择合适的电源
- **摄像头**: 通过USB供电

## 安全注意事项

1. 确保电源电压正确
2. 检查连线是否牢固
3. 避免短路
4. 在断电情况下进行连线
